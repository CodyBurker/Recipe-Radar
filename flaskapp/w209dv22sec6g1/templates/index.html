About

The goal of this application is visualizing flavor penetrations of cuisines to allow users to see trends among flavors, macronutrients and ingredient based specifics.

Target Audience

Foodies, Nutritionists, Dieticians, Home Cooks, Chefs, & more !

Users can use our site and visualizations to gain an understanding of global flavor profiles, tendencies with respect to dominating macronutrients and even dominating ingredients!

Data Source

The Data used for the visualization in this website is sourced from Spoonacular.

Spoonacular is a small team of PHD Computer Scientists who have curated food and recipe related information into a database. The information is then exposed via REST and has many query parameters that run through semantic search to output results.

The following steps were taken to accumulate data from Spoonacular.

  1. 1. Procure recipes based on cuisines.
  2. 2. For each recipe, call additional endpoints like flavor/nutrition/ingredients.
  3. 3. Flatten the JSON response objects from each endpoint.
  4. 4. Join relevant data into dataframes to use for visualizations.
